## 2) MQTT / MQTTS 命令错误码（MQTT用户手册 V6.8.5）

来源：`MQTT用户手册_V6.8.5.pdf` 第 36 页

| 错误码 | 释义 |
|---:|---|
| 600 | 未知错误 |
| 601 | 无效参数 |
| 602 | 未连接或连接失败 |
| 603 | 正在连接 |
| 604 | 已经连接 |
| 605 | 网络错误 |
| 606 | 存储错误 |
| 607 | 状态错误 |
| 608 | DNS错误 |
| 609~649 | 保留 |

补充（连接状态枚举，非错误码，但常用于排查）：
- `AT+MQTTCONN` / `+MQTTURC: "conn"` 中的 `<conn_state>` 在手册中枚举为：
  - 0：连接成功
  - 1：正在重连
  - 2：断开：客户端断开
  - 3：断开：服务器拒绝
  - 4：断开：服务器断开
  - 5：断开：ping超时
  - 6：断开：网络异常
  - 255：断开：未知错误
